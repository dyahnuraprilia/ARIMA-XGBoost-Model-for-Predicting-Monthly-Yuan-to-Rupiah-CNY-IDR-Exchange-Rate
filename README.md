# ARIMA-XGBoost-Model-for-Predicting-Monthly-Yuan-to-Rupiah-CNY-IDR-Exchange-Rate
Proyek analisis deret waktu untuk memprediksi nilai tukar bulanan Yuan terhadap Rupiah (CNY/IDR) menggunakan model ARIMA (0,1,0) dan XGBoost Regression. Berdasarkan metrik evaluasi (MSE &amp; R²), XGBoost memberikan performa prediksi dan validasi yang lebih akurat untuk data kurs
